## Notes

- `results/metrics/` contains the SASHTS benchmark/ablation evidence and the multi-dataset trace/source-detection tables.
- `results/model/` contains the full PyTorch two-layer weighted mean GraphSAGE metrics, tuning tables, and train/validation/test gap summaries.
- `results/audit/` contains Phase 04 metadata and selected hyperparameters.
- `results/orders/` contains intermediate or hard-to-classify artifacts that should be reviewed before final archival; currently this includes `trace_candidate_features.csv`.
- `scripts/28_phase04_torch_hyperparameter_tuning.py` runs validation-CV hyperparameter search over `hidden_dim`, `dropout_1`, `dropout_2`, `lr`, and `weight_decay`. Run it through `scripts/run_phase04_torch_hyperparameter_tuning.cmd` in a CMD environment where `torch` is installed.
- `scripts/29_phase04_gap_summary.py` generates train-test and validation-test gap summaries from the full PyTorch multi-seed/fold metrics.
